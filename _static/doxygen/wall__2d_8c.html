Load 2D wall data and prepare parameters.

This function assumes offload data is already finished and the offload array is allocated and initialized with values

&(*offload_array)[0] = Wall polygon R coordinates &(*offload_array)[n] = Wall polygon z coordinates

Since this data requires no initialization, the only thing this function does is that it prints some values as sanity check.

Check if coordinates are within 2D polygon wall.

This function checks if the given coordinates are within the walls defind by a 2D polygon using a modified axis crossing method [1]. Origin is moved to the coordinates and the number of wall segments crossing the positive r-axis are calculated. If this is odd, the point is inside the polygon.

[1] D.G. Alciatore, R. Miranda. A Winding Number and Point-in-Polygon Algorithm. Technical report, Colorado State University, 1995. http://www.engr.colostate.edu/~dga/dga/papers/point_in_polygon.pdf

Parameters
rr coordinate
zz coordinate
w2D wall data structure
